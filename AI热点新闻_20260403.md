# AI热点新闻
> 更新日期：2026年04月03日

## 一、AI技术

**1. Google DeepMind 发布 Gemma 4 开源模型，"每字节最强"** Google 于4月2日推出新一代开源模型系列 Gemma 4，专为高级推理和智能体工作流设计，包含 2B/4B/26B MoE/31B Dense 四种规格，采用 Apache 2.0 许可证。31B 模型在 Arena AI 排行榜位列开源第三，性能超越20倍参数量的竞品。 [[1]](https://blog.google/innovation-and-ai/technology/developers-tools/gemma-4/)

**2. 阿里千问发布 Qwen 3.6-Plus，国产编程模型新标杆** 阿里4月2日发布新一代大语言模型 Qwen 3.6-Plus，编程能力超越 GLM-5、Kimi-K2.5 等2-3倍参数量的国产模型，接近 Claude 水平。支持100万 Token 上下文窗口，已针对 OpenClaw、Claude Code 等主流 Agent 框架深度优化。 [[2]](https://www.chinanews.com.cn/cj/2026/04-02/10597595.shtml)

**3. Anthropic Claude Code 源码意外泄露，51万行代码被扒光** 3月31日，Anthropic 发布 Claude Code v2.1.88 时忘记删除 npm 包中的 Source Map 文件，导致1906个源文件、51万行 TypeScript 代码被完整还原。社区7小时内发现8个未发布功能、26个隐藏指令和6层安全架构。 [[3]](https://36kr.com/p/3747481076417289)

**4. 全球AI观察：智能体从"会聊天"迈向"会干活"** 科技日报4月1日发文总结3月全球AI动态，指出以 OpenClaw 为代表的 AI Agent 正从对话交互加速转向自主执行任务，多国加大AI基础设施战略投资。同时警示AI系统"过度迎合"用户的谄媚机制风险。 [[4]](https://www.stdaily.com/web/gdxw/2026-04/01/content_496125.html)

**5. Anthropic 内部最强模型 Claude Mythos 意外曝光** Fortune 3月31日报道，Anthropic 因内容管理系统配置错误，意外公开约3000份未发布博客资料，其中披露了代号"Mythos"（又名 Capybara）的下一代模型——定位在 Opus 之上的全新第四层级，被称为"迄今最强AI模型"。 [[5]](https://fortune.com/2026/03/31/anthropic-source-code-claude-code-data-leak-second-security-lapse-days-after-accidentally-revealing-mythos/)
