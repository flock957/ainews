# AI热点新闻
> 更新日期：2026年04月04日

## 一、AI技术

1. **Anthropic即将发布的"Mythos"模型引发网络安全担忧** — Anthropic意外泄露了其下一代AI模型Claude Mythos的细节，该模型在编程、学术推理和网络安全等测试中大幅超越Claude Opus 4.6，被称为"能力的阶梯式跃升"。Anthropic已私下警告政府官员，该模型能以前所未有的速度发现和利用漏洞，可能使大规模网络攻击更加容易。 [[1]](https://edition.cnn.com/2026/04/03/tech/anthropic-mythos-ai-cybersecurity)

2. **微软发布三款基础AI模型，直接挑战OpenAI和谷歌** — 微软AI团队（由Mustafa Suleyman领导）发布了MAI-Transcribe-1（语音转文字，支持25种语言，速度为Azure Fast的2.5倍）、MAI-Voice-1（1秒生成60秒音频）和MAI-Image-2（图像生成速度提升2倍），已通过Microsoft Foundry开放使用，定价低于谷歌和OpenAI同类产品。 [[2]](https://techcrunch.com/2026/04/02/microsoft-takes-on-ai-rivals-with-three-new-foundational-models/)

3. **阿里巴巴发布Qwen3.6-Plus，百万token上下文对标Claude Code** — 阿里通义千问4月2日发布新一代大模型Qwen3.6-Plus，默认支持100万token上下文窗口，具备原生多模态能力，可从截图和设计稿生成代码。该模型在SWE-bench上接近Claude Opus 4.5，输入价格仅2元/百万token，将整合进悟空企业平台和通义APP。 [[3]](https://www.caixinglobal.com/2026-04-02/alibaba-releases-qwen-36-plus-ai-model-with-enhanced-coding-capabilities-102430395.html)

4. **Allen Institute发布开源Web Agent MolmoWeb，8B模型超越GPT-4o** — AI2发布了开源浏览器自动化Agent MolmoWeb，通过截图理解网页（而非解析HTML），执行点击、输入、滚动等操作完成任务。8B参数版本在关键网页导航任务上超越GPT-4o等大型闭源模型，模型权重、训练数据和评估工具均已开源。 [[4]](https://www.geekwire.com/2026/ai2-releases-open-source-web-agent-to-rival-closed-systems-from-openai-google-and-anthropic/)

5. **Anthropic指控DeepSeek、Moonshot、MiniMax使用2.4万假账号窃取Claude能力** — Anthropic公开指控三家中国AI实验室通过约2.4万个假账号与Claude进行超过1600万次对话，系统性地蒸馏其模型能力。此事件引发了关于AI模型保护、代理访问和美国出口管制的新一轮讨论。 [[5]](https://venturebeat.com/technology/anthropic-says-deepseek-moonshot-and-minimax-used-24-000-fake-accounts-to)

## 二、AI智能终端

1. **全球首个人形机器人格斗联赛URKL开放报名** — 由EngineAI发起的URKL（Ultimate Robot Knock-out Legend）于4月3日正式面向全球高校、企业和研究机构开放报名，采用EngineAI T800全尺寸人形机器人作为统一平台，聚焦运动控制和感知决策算法竞争，冠军将获得价值约1000万元人民币的纯金冠军腰带。 [[6]](https://www.globenewswire.com/news-release/2026/04/03/3267997/0/en/The-World-s-First-Humanoid-Robot-Free-Combat-League-URKL-Officially-Opens-Global-Registration.html)

2. **Nothing计划推出AI智能眼镜和AI耳机** — 据Bloomberg报道，Nothing正在开发配备摄像头、麦克风和扬声器的AI智能眼镜（预计2027上半年发布），以及一款AI耳机（预计今年内发布）。CEO裴宇此前对眼镜持保留态度，但已转向多设备生态战略，将与Meta、三星等展开竞争。 [[7]](https://techcrunch.com/2026/04/01/nothings-ai-devices-plan-reportedly-contains-smart-glasses-and-earbuds/)

3. **德国Neura Robotics获10亿欧元融资，亚马逊和高通领投** — 欧洲人形机器人初创公司Neura Robotics以40亿欧元估值完成约10亿欧元融资，投资方包括亚马逊、高通、博世和卡塔尔王室成员。这是欧洲机器人领域最大单笔投资之一，公司专注于可与人类协作的认知型人形机器人，应用于制造、物流和医疗领域。 [[8]](https://siliconangle.com/2026/03/04/humanoid-robot-maker-neura-robotics-reportedly-raising-1-2b-funding/)

4. **小鹏成立Robotaxi事业部，计划2026年推出三款无人出租车** — 小鹏汽车3月底正式成立独立Robotaxi事业部，计划今年推出5座、6座、7座三款L4级无人出租车，搭载4颗图灵AI芯片（算力达3000 TOPS），采用纯视觉方案不依赖激光雷达。已与高德达成生态合作，下半年启动载客示范运营。 [[9]](https://cleantechnica.com/2026/03/30/xpeng-serious-about-robotaxis-creates-distinct-division/)

5. **三星首次公布AI智能眼镜细节，2026年内发售** — 三星在MWC 2026上首次披露AI智能眼镜规格：配备眼部水平摄像头捕捉用户视野，与手机协同处理AI任务，手机端负责重计算。三星采用"眼镜做感知、手机做大脑"的分工策略，挑战目前占据82%市场份额的Meta Ray-Ban眼镜。 [[10]](https://www.cnbc.com/2026/03/06/samsung-ai-smart-glasses-first-details-specs-release-date.html)
