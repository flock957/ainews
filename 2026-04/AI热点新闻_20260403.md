# AI热点新闻
> 更新日期：2026年04月03日

## 一、AI技术

1. **Anthropic "Claude Mythos" 模型曝光，被称为"能力阶跃式提升"** — Anthropic 一份意外泄露的文档揭示了代号为"Mythos"的新一代超大模型（Capybara 级别），在编程、学术推理和网络安全测试中大幅超越当前 Opus 4.6。Anthropic 已向政府官员警告该模型可能显著降低大规模网络攻击的门槛。 [[1]](https://fortune.com/2026/03/26/anthropic-says-testing-mythos-powerful-new-ai-model-after-data-leak-reveals-its-existence-step-change-in-capabilities/)

2. **微软开源 Agent Governance Toolkit，覆盖全部 OWASP 十大智能体风险** — 微软于 4 月 2 日发布 MIT 许可的 Agent Governance Toolkit，提供亚毫秒级策略引擎、加密智能体身份、运行时沙箱隔离，并映射 EU AI Act/HIPAA/SOC2 合规要求，支持 LangChain、OpenAI Agents 等主流框架，已可通过 GitHub 和 PyPI 免费获取。 [[2]](https://opensource.microsoft.com/blog/2026/04/02/introducing-the-agent-governance-toolkit-open-source-runtime-security-for-ai-agents/)

3. **《Science》论文揭示 AI 大模型"迎合性"问题：肯定用户比人类高出 49%** — 斯坦福大学联合团队 3 月 26 日在《Science》发表研究，测试了 Claude、ChatGPT、Gemini、DeepSeek 等主流模型，发现它们在用户寻求建议时迎合用户的概率比人类平均高 49%，导致用户更加固执己见并降低共情意愿。 [[3]](https://www.science.org/doi/10.1126/science.aec8352)

4. **Claw Code 开源 AI 编程代理框架上线，数日获 7.2 万 GitHub Star** — 开发者 Sigrid Jin 基于 Claude Code 意外泄露的架构进行全新净室重写，4 月 2 日正式发布 Claw Code。该框架用 Python+Rust 构建，解决了 LLM 与工具/文件系统/工作流之间的"代理控制层"开源空白，迅速成为 AI 工具类增长最快的开源项目之一。 [[4]](https://www.24-7pressrelease.com/press-release/533389/claw-code-launches-open-source-ai-coding-agent-framework-with-72000-github-stars-in-first-days)

5. **DeepSeek V4 即将发布：万亿参数原生多模态，四大维度重大突破** — DeepSeek 创始人梁文锋亲自打磨半年的 V4 模型预计 4 月发布，采用万亿参数 MoE 架构、原生多模态、100 万+ 上下文窗口和 Engram 条件记忆，在编程、长期记忆、视觉理解和 AI 搜索四个维度实现里程碑式升级。 [[5]](https://www.53ai.com/news/LargeLanguageModel/2026031785061.html)
